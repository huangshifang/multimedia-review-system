# 多媒体作品评审系统 — 用户手册

## 系统简介

多媒体作品评审系统用于组织多媒体作品比赛评选，支持三种角色协同工作：

- **管理员 (ADMIN)**：创建比赛、管理参评人、分配评委、控制比赛流程
- **评委 (JUDGE)**：匿名评分、在线预览作品
- **选手 (PARTICIPANT)**：查看参与的比赛、上传作品文件

## 访问地址

浏览器打开：**http://localhost:5173**

## 一、管理员操作指南

### 1.1 登录

| 用户名 | 密码 |
|--------|------|
| admin | admin123 |

### 1.2 创建比赛

1. 登录后进入仪表盘，点击 **"创建比赛"** 按钮
2. 填写信息：
   - **比赛名称**：必填
   - **描述**：可选
   - **名次数量**：选择 1-10 名
   - **总参赛人数**：填写预计参赛总人数
   - **评分修改时限**：评委提交评分后几分钟内可修改（默认 10 分钟）
3. 设置各名次名额：
   - 前 N-1 名的名额手动填写
   - **最后一名名额自动计算** = 总参赛人数 - 前 N-1 名名额之和
4. 点击确定创建

### 1.3 配置比赛

比赛创建后处于 **草稿** 状态。点击对应比赛的 **"配置"** 按钮进入配置页。

**添加参评人：**
1. 切换到"参评人"标签页
2. 点击"添加参评人"
3. 填写姓名、部门
4. **关联选手账号**：下拉选择已有的选手账号（可选，关联后选手可自主上传作品）
5. 添加后可继续上传参评人的作品文件

**分配评委：**
1. 切换到"评委"标签页
2. 点击"分配评委"
3. 从列表中选择评委用户（多选）
4. 确定后即可分配

### 1.4 控制比赛流程

- **开始比赛**：配置完成后，在仪表盘点击 **"开始"** 按钮，比赛进入"评分中"状态
- **结束比赛**：评分收集完毕后，点击 **"结束"**，系统自动生成最终排名

### 1.5 查看报表

比赛进入评分中或已结束后，点击 **"查看报表"** 可看到实时排名，点击 **"打印排名表"** 可弹出正式排名表并打印。

## 二、评委操作指南

### 2.1 登录

| 用户名 | 密码 |
|--------|------|
| judge1 | judge123 |
| judge2 | judge123 |

也可通过注册页面自行注册评委账号。

### 2.2 评分流程

1. 登录后在仪表盘找到状态为 **"评分中"** 的比赛
2. 点击 **"评分"** 按钮进入评分页
3. 页面展示所有参评人及其作品文件（图片/视频/音频/文档均可在线预览）
4. 为每位参评人打分（0-100 分，支持一位小数）
5. 点击 **"提交评分"** 按钮

**重要规则：**
- 提交后 **10 分钟内可修改**（时长由管理员配置）
- 超时后评分自动锁定，**不可再修改**
- 评分期间评委之间**互不可见评分**

### 2.3 排名算法

- 3 位及以上评委：**去掉最高分和最低分**，取剩余分数平均
- 不足 3 位评委：所有分数直接取平均

### 3.1 登录

| 用户名 | 密码 |
|--------|------|
| player1 | player123 |

也可通过注册页面选择"选手"角色自行注册。

### 3.2 上传作品

**前提：** 管理员已创建比赛并将你添加为参评人（关联了你的账号），且比赛已进入"评分中"或之后的状态。

1. 登录后在仪表盘找到已关联的比赛
2. 点击 **"我的作品"** 按钮进入工作台
3. 看到你的参赛条目卡片
4. 点击 **"上传作品"** 按钮选择文件

**支持格式：**

| 类型 | 扩展名 | 大小限制 |
|------|--------|----------|
| 图片 | jpg, jpeg, png, gif, bmp, webp, svg | 20 MB |
| 文档 | txt, doc, docx, pdf, xls, xlsx, ppt, pptx | 10 MB |
| 视频 | mp4, mov, avi, mkv, wmv, flv, webm | 500 MB |
| 音频 | mp3, wav, aac, flac, ogg, wma | 50 MB |

> 系统自动拦截可执行文件（exe、bat、脚本等），保障安全。

5. 上传后可在线预览、下载或删除文件

```
草稿 (DRAFT)

  │
  ▼  [管理员点击"开始"]
  │

  │  评委匿名评分，选手可上传稿件
  │
  ▼  [管理员点击"结束"]
  │
已结束 (FINISHED)
  │  生成最终排名，可查看报表、打印排名表
```

## 六、常见问题

**Q: 上传文件提示"不支持的文件类型"？**
A: 系统仅支持图片/文档/视频/音频格式，可执行文件（exe、脚本等）禁止上传。请检查文件格式是否在支持列表中。

**Q: 选手登录后看不到比赛？**
A: 需要管理员在比赛配置中把你添加为参评人并关联你的账号，且比赛需处于"评分中"或之后的状态。

**Q: 评委看不到"评分"按钮？**
A: 确认：1) 比赛状态为"评分中"；2) 你已被管理员分配到该比赛。

**Q: 为什么评分后不能修改？**
A: 超出评分修改时限后系统自动锁定。时限由管理员创建比赛时设置（默认 10 分钟）。

**Q: 名次是如何分配的？**
A: 按平均分从高到低排序，依次分配到第 1 名、第 2 名......每名次名额由管理员在创建比赛时设定。最后一名名额自动计算 = 总参赛人数 - 前面名次名额之和。

**Q: 同分怎么处理？**
A: 系统自动检测同分情况，管理员可发起复评。
